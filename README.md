## Installation

1. Clone

   ```
   git clone --recursive https://github.com/rwxdan/.dotfiles
   ```

2. Get `stow`'ned don't `cp`
   ```
   cd .dotfiles
   stow . -t ~/
   ```
3. ❓❓❓
4. Profit.
